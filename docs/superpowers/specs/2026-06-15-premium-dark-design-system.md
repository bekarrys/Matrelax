# Дизайн-система MATRELAX: Premium Dark

**Дата:** 2026-06-15
**Статус:** согласован, готов к написанию плана

## Контекст и цель

Текущий интерфейс выглядит «шаблонно» не из-за отсутствия дизайн-системы — она
есть (тёмная тема, токены в [globals.css](../../../client/src/styles/globals.css),
шрифт Golos Text, pill-кнопки, анимации). Проблема в **исполнении**: плоский
почти-чёрный фон без слоёв, один резкий электрик-синий акцент, серые заглушки
вместо фото, отсутствие глубины и тёплых теней.

**Цель:** поднять восприятие до «премиум», оставаясь в тёмной теме. Не флип темы,
а её обогащение: слои поверхностей, благородный акцент + золотой нюанс, тени-свечения,
hover-глубина, выраженный бренд.

**Охват:** обе поверхности на единых токенах — публичная витрина и панель
персонала/CRM.

### Что НЕ входит

- Смена шрифтового семейства (Golos Text остаётся — лучшая кириллица, чем Inter/Manrope).
- Градиентные кнопки (читаются «туториально» — используем плотные кнопки + свечение).
- Внешние анимационные библиотеки (framer-motion и т.п.) — только CSS-keyframes.
- Реальная продуктовая фотосъёмка (отдельная задача; здесь — только аккуратные
  плейсхолдеры и зоны под фото).
- Реструктуризация компонентов/роутинга — только визуальный слой.

## Принятые решения

| Тема | Решение |
|------|---------|
| Тема | Остаёмся тёмными; «премиум-дарк» вместо плоского чёрного |
| Шрифт | Golos Text (без смены семейства; правим типошкалу) |
| Акцент | Мягкий электрик-индиго `#6D6BF6` вместо резкого `#2563eb` |
| Премиум-нюанс | Champagne-золото `#D4B483` точечно (цена, бейджи) |
| Кнопки | Плотные (акцент/ink) + тень-свечение; без градиентов |
| Анимации | Существующие CSS-keyframes + staggered-вход карточек |
| Охват | Обе поверхности на единых токенах |
| Стратегия | Центр тяжести — токены; компоненты уже на `var(--…)` |

## Токены (правка `client/src/styles/globals.css`)

### Поверхности и текст
```css
--bg:             #0B0B0E;  /* теплее/глубже плоского #0a0a0a */
--surface:        #141418;  /* карточки — видимая ступень от фона */
--surface-2:      #1C1C22;
--surface-3:      #26262E;
--text-primary:   #F4F4F5;
--text-secondary: #9A9AA2;  /* было #888888 — поднимаем читаемость */
--text-tertiary:  #5C5C66;
--border:         rgba(255,255,255,0.07);
--border-strong:  rgba(255,255,255,0.14);
```

### Акцент и премиум-нюанс
```css
--accent:       #6D6BF6;
--accent-hover: #5A57F2;     /* чуть темнее акцента */
--accent-glow:  rgba(109,107,246,0.35);
--accent-tint:  rgba(109,107,246,0.12);
--gold:         #D4B483;
--gold-tint:    rgba(212,180,131,0.12);
```

### Тени (глубина вместо плоскости)
```css
--shadow-sm:    0 2px 12px rgba(0,0,0,0.45);
--shadow-md:    0 8px 32px rgba(0,0,0,0.55);
--shadow-glow:  0 4px 24px var(--accent-glow);   /* для CTA */
```

### Радиусы и прочее
Существующие сохраняются (`--radius: 16px`, `--radius-sm: 10px`,
`--radius-full: 999px`). Добавить `--radius-lg: 24px` для hero-блоков.
Статусные цвета (`--status-progress/ready/delivery/delivered`) сохраняются.

## Компоненты

### Карточка товара (`ProductCard`)
- Фон `--surface`, тонкая верхняя highlight-граница `rgba(255,255,255,0.06)`.
- Базовая тень `--shadow-sm`.
- Hover: `border` → `--border-strong`, `transform: translateY(-4px)`,
  тень → `--shadow-md` + лёгкое свечение.
- Цена — акцентом: число в `--gold` или индиго-pill; «от N ₸» уверенным весом.
- Зона под фото — аккуратный градиентный плейсхолдер (не пустой серый),
  иконка-глиф приглушённым тоном.
- Вход списка — staggered (`animation-delay` по индексу, на базе `cardAppear`).

### Hero (главная)
- Многоступенчатый радиальный градиент (язык экрана `Login` — уже работает),
  радиус `--radius-lg`.
- Крупный display-заголовок; eyebrow «сделано в казахстане» в `--accent`.
- Достаточный контраст текста (текущий серый подзаголовок слишком тусклый).

### Кнопки (`globals.css` .btn-*)
- `.btn-primary`: фон `--accent`, hover/active `--accent-hover`, тень `--shadow-glow`.
- `.btn-ink` (новая/переименование secondary): фон `--surface-2`, граница `--border`.
- Сохранить `btn-press` (scale 0.97 на :active).

### Шапка/бренд (`Layout`)
- Wordmark **MATRELAX** + bed-глиф (уже есть на `Login`) вынести в шапку.
- Разделитель/индикатор роли для панели персонала.
- Активный пункт навигации — подчёркивание/тон акцентом, не только bold.

### Панель/CRM
- Те же токены, плотнее: тёмный shell, sidebar на `--surface`/`ink`.
- Таблицы: разделители `--border`, hover-строки `--surface-2`.
- Статус-чипы: текст статусного цвета на его `-bg` тинте (уже определены).

## Стратегия внедрения (низкий риск)

1. **Токены:** обновить значения в `globals.css` (центр тяжести — большинство
   компонентов уже ссылаются на `var(--…)`).
2. **Аудит хардкода:** найти захардкоженные hex-цвета в `*.css` компонентов
   (напр. `#1e1e1e` в skeleton) → заменить на токены.
3. **Точечная глубина:** поднять `ProductCard`, hero (`Home`), `.btn-*`, `Layout`.
4. **Проверка обеих поверхностей:** прогнать витрину и панель, сверить контраст
   и читаемость на реальных страницах (скриншоты).

## Критерии готовности

- Фон имеет различимые слои (фон/карточка/поднятый блок), не «плоский чёрный».
- Карточки товара имеют тень и hover-глубину; цена выделена (золото/акцент).
- Hero контрастный, с глубиной градиента и брендовым заголовком.
- Кнопки — плотные со свечением, без градиентов; press/hover работают.
- Бренд MATRELAX виден в шапке обеих поверхностей.
- Нет захардкоженных «старых» цветов (`#2563eb`, `#888888` и т.п.) в компонентах.
- Текстовый контраст вторичного текста читаем (не тусклее `--text-secondary`).
